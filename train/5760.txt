Tudor DICOM Tools are a Java library to perform high-level DICOM operations. The library includes stand-alone applications and ImageJ plugins. Functionality includes reading and writing to DICOM files or DICOMDIR filesets, network transfers including client and PACS server functions, and image viewing and analysis. Most of the functions can be used as ImageJ macros. Included are the library and sample applications built from it, such as the Tudor DICOM Viewer.
