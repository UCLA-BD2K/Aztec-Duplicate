Oviyam is a web-based DICOM viewer that requires no special client software or downloads of any kind. It is installed as a component on a PACS server and delivers image files as standard HTML5 to any compatible web browser. Oviyam uses standard Web protocols to allow PACS querying, and retrieval and display of selected series or studies. It is packaged and deployed on the JBoss application framework, and can communicate with any WADO-compatible PACS server
